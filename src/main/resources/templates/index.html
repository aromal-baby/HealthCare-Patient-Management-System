About This Project

This Healthcare Patient Management System was developed as part of the M605 Advanced Databases course at Gisma University of Applied Sciences. It demonstrates database design principles, hybrid database architecture, and software engineering best practices.

The system implements a comprehensive healthcare database using both MySQL for structured data (patient information, appointments, staff) and MongoDB for semi-structured/unstructured data (medical records, test results, patient history). This hybrid approach showcases the strengths of each database paradigm and provides an optimal solution for healthcare data management.
